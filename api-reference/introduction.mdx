## Autenticação

Todas as requisições à API devem ser autenticadas usando **API Key** no formato `ApiKey {sua_chave}`.

### Formato do Header

```bash
Authorization: ApiKey sua_api_key_aqui
```

### Como obter sua API Key

1. Faça login no painel do Simplo
2. Vá para Configurações > Integrações > API Keys
3. Clique em "Gerar Nova API Key"
4. Copie e guarde sua chave em local seguro

### Exemplo de Uso

```bash
curl -X GET "https://api.simplo.com/api/v1/subscriptions" \
  -H "Authorization: ApiKey sua_api_key_aqui" \
  -H "Content-Type: application/json"
```

### Erros de Autenticação

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    Retorna quando a API key não é fornecida ou está em formato inválido.
  </Accordion>
  <Accordion title="404 Not Found">
    Retorna quando a API key é inválida ou a conta está inativa.
  </Accordion>
</AccordionGroup>

## Formatos de Resposta

A API utiliza o formato `application/problem+json` para respostas de erro:

```json
{
  "type": "https://problems-registry.smartbear.com/validation-error/",
  "status": 422,
  "title": "Validation Error",
  "detail": "O payload da requisição contém erros de validação",
  "errors": [
    {
      "detail": "Customer ID é obrigatório",
      "pointer": "/subscription/customer_id"
    }
  ]
}
```

## Versão da API

A versão atual é `v1` e está disponível em:
```
https://api.simplo.com/api/v1/
```
